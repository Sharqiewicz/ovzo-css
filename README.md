## Usage

There are 3 ways to get started with OvzoCSS:

**Install with NPM**

```shell
npm install ovzocss
yarn add ovzocss
```

**Install manually**

[Download OvzoCSS](to be defined) and link `/css/ovzo.css` in the `<head>` of your website.

```html
<link rel="stylesheet" href="css/ovzo.css">
```

**Install from CDN**

Use CDN: [jsDelivr CDN](to be defined)

```html
<link rel="stylesheet" href="to be defined">
```
